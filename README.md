# StudyHub Planner

## 1. Название проекта

**StudyHub Planner** — веб-приложение для планирования учебных задач студентов.

---

## 2. Описание проекта

StudyHub Planner — это приложение для управления учебными заданиями.  
Оно помогает студентам отслеживать дедлайны, приоритет задач и прогресс выполнения.

Основные функции:

- добавление задач (название, предмет, дедлайн, приоритет);
- отображение списка задач;
- фильтрация по статусу (все / активные / выполненные);
- фильтрация по предметам;
- отметка задачи как выполненной;
- удаление задач;
- сохранение данных в браузере.

Целевая аудитория:

- студенты;
- учащиеся;
- начинающие разработчики React.

---

## 3. Установка и запуск

### Требования

- Node.js версии 16 или выше
- npm
- современный браузер

### Шаги установки

1. Клонировать репозиторий:

git clone https://github.com/ВАШ_АККАУНТ/studyhub-planner.git

2. Перейти в папку проекта:

cd studyhub-planner

3. Установить зависимости:

npm install

4. Запустить приложение:

npm run dev

5. Открыть в браузере:

http://localhost:5173

---

## 4. Примеры использования

### Добавление задачи

1. Ввести название задания
2. Выбрать предмет
3. Указать дедлайн
4. Нажать кнопку «Добавить»

Задача появится в списке.
